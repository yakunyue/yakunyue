## 关于小链家

这是一个链家二手房交易记录的汇总网站，数据大多收集自 [链家网](www.lianjia.com) 。

网站依靠一些图表工具，直观的展示任意城市、区县、街区、小区的成交记录和价格变动曲线。

我开发它的目的是为计划买房的人提供更直观的价格参考。
